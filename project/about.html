About Recipe Explorer

Recipe Explorer is a simple web application designed to help people find recipes based on ingredients, cooking time and cuisine. It is built for anyone who wants quick inspiration without complicated steps.

Why this site exists

Many people struggle with deciding what to cook using the ingredients they already have. Recipe Explorer makes that easier by letting users browse and filter recipes, save favorites and view clear instructions.

Accessibility

The site is designed so that as many people as possible can use it comfortably. Pages use clear structure, readable colors and work with keyboards and screen readers.

  • Semantic HTML is used for clear page structure.
  • Navigation and buttons are keyboard accessible.
  • Images include descriptive text for screen readers.

Technology Used

Recipe Explorer was built using core web technologies taught in this class without external frameworks.

  • HTML, CSS and JavaScript.
  • Local JSON data loaded with JavaScript.
  • Local storage to save user preferences.

About the Project

This website was created as an individual project to practice responsive design, accessibility and dynamic content using modern JavaScript techniques.
